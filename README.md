# soc-phishing-investigation-lab
Hands-on SOC Analyst phishing investigation lab demonstrating alert triage, phishing detection, true/false positive classification, incident reporting, and security analysis.
## Overview
This repository documents my hands-on SOC Analyst phishing investigation lab completed as part of cybersecurity training.
The objective was to investigate phishing-related alerts, identify True Positives (TP) and False Positives (FP), and follow a structured incident response workflow.
**Note:** This repository documents my learning process and does **not** include TryHackMe flags, answers, or walkthroughs.

---

- Analyze phishing alerts
- Perform alert triage
- Differentiate True Positives and False Positives
- Investigate suspicious emails

## Investigation Workflow

1. Receive security alert
2. Review alert details
3. Verify sender information
4. Inspect URLs and attachments
5. Determine TP or FP
6. Document findings
7. Recommend remediation
